# Industrial Website Strategic Audit\
\
Perform a complete strategic audit of any industrial website.\
\
Unlike a site teardown, the goal is not simply to understand how the website was built.\
\
The goal is to determine:\
\
- Why it works\
- Why it fails\
- What should be copied\
- What should be improved\
- What should be ignored\
- How a better version should be built\
\
The output should be useful for:\
\
- CEOs\
- Founders\
- Product managers\
- Marketing teams\
- Industrial manufacturers\
- Website designers\
- Developers\
\
The final audit should be detailed enough that a complete website redesign can be planned and executed from it.\
\
---\
\

\
Activate when the user wants to:\
\
- Analyze an industrial website\
