Every high-impact system depends on a content model that does not yet exist. Building pages first = re-hardcoding. Building the model first = every later system is data population, not engineering.

## 4. Ranked backlog (by leverage тАФ what unlocks the most)

1. **Content Relationship Model (catalog + data API)** тАФ KEYSTONE. Unlocks #1,5,6,7,8,10,11.
2. **Application Layer** тАФ first-class application entities + data-driven application pages. The biggest IA unlock and the cross-market connective tissue; first real consumer of the model.
3. **Shared design-system stylesheet / chrome** тАФ velocity multiplier for every new page type.
4. **Product Architecture + Products hub** тАФ families/products as catalog entities; shared catalog component.
5. **Market-page template тЖТ remaining 7 markets + data-driven nav.**
6. **Resource layer + Knowledge Center.**
7. **Conversion / Lead-gen (RFQ + sample request).**
8. **Search over the catalog.**

## 5. Recommended implementation sequence (this milestone first)

**Milestone A (NOW): Content Model + Application Layer.**
- `assets/data/deon-catalog.js` тАФ entity graph (markets, segments, first-class applications incl. cross-market ones, product families, products [reuse the 24 SKUs], resources).
- `assets/deon-data.js` тАФ `window.DEON` query + URL helpers.
- `assets/deon-base.css` тАФ shared design-system sheet so new pages don't duplicate inline CSS.
- `application.html` + renderer тАФ one data-driven template that renders ANY application by `?app=<id>`, with breadcrumb, challenges, markets-used-in, product families, related applications, resources, contextual CTA тАФ all cross-linked from the graph.
- Cross-link Electronics Explorer тЖТ application pages.

**Then re-audit** and proceed to Products hub (Milestone B), market template + data-driven nav (C), resources/knowledge (D), lead-gen (E), search (F).

**Decision:** build Milestone A now. The catalog is the keystone; the Application Layer proves it and delivers the cross-linking + IA depth that separates a brochure from a Tesa-grade ecosystem.
